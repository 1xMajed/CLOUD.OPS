المسار الكامل

١٤ خطوة من الصفر إلى مهندس كلاود

هذا الترتيب مبني على أفضل الممارسات في سوق العمل، وهو التسلسل المنطقي الذي يظهر فعلياً في إعلانات وظائف مهندس الكلاود.

1
أساسي
أساسيات تقنية المعلومات، الكلاود، TCP/IP، وDevOps
IT, Cloud, TCP/IP, and DevOps Fundamentals
نقطة الانطلاق. هنا تتعرّف على المفاهيم الأساسية: ما هو الكلاود؟ كيف تتواصل الشبكات؟ ما معنى DevOps؟ لا تحتاج خبرة سابقة لهذه الخطوة.
2
أساسي
مقدمة لخدمات AWS الأساسية للمبتدئين
AWS Core Services Introduction
أول تعارف عملي مع AWS Console. ستتعلم EC2 وS3 وIAM وVPC على مستوى مبسّط.
3
أساسي
معسكر Linux للمبتدئين
Linux Bootcamp for Beginners
كل سيرفر كلاود تقريباً يعمل بـ Linux. تعلّم سطر الأوامر، الصلاحيات، إدارة الملفات والعمليات.
4
أساسي
إتقان Bash Scripting
Bash Scripting Mastery
بعد أن تتعلّم أوامر Linux، تبدأ بكتابة سكريبتات تؤتمت المهام المتكررة. أول خطوة نحو الأتمتة.
5
أساسي
إتقان Git
Git Mastery
من هذه النقطة فصاعداً، كل كود تكتبه يجب أن يكون تحت التحكم بالإصدارات. ابدأ بناء سجلّك الآن.
6
أساسي
أتمتة AWS باستخدام Python
AWS Cloud Automation Using Python
الآن تربط Python بـ AWS عبر مكتبة boto3. تتعلم كتابة سكريبتات تتحكم بموارد AWS تلقائياً.
7
أساسي
YAML للمبتدئين
YAML for Beginners
خطوة قصيرة لكنها ضرورية. YAML هي اللغة التي تُكتب بها إعدادات Docker وKubernetes وCI/CD.
8
أساسي
إتقان Docker
Docker Mastery
الحاويات هي الطريقة الحديثة لتشغيل التطبيقات. تتعلم بناء صور Docker، تشغيل حاويات، وربطها ببعضها.
9
أساسي
إتقان Kubernetes
Kubernetes Mastery
بعد إتقان حاوية واحدة، تتعلم إدارة مئات الحاويات معاً بـ Kubernetes. يستخدمه معظم الشركات الكبيرة.
10
شهادة
شهادة: Terraform Certified Associate
Terraform Certified Associate
أول شهادة في مسارك. Terraform هي اللغة التي يكتب بها مهندس الكلاود البنية التحتية ككود.
11
شهادة
شهادة: AWS Solutions Architect Associate
AWS Solutions Architect Associate (SAA-C03)
الشهادة الأكثر طلباً في سوق الكلاود عالمياً. تُثبت قدرتك على تصميم بنية تحتية كاملة على AWS.
12
اختر واحدة
CI/CD باستخدام Jenkins & GitHub Actions — أو — GitLab CI/CD
12-A) Jenkins + GitHub Actions | 12-B) GitLab CI/CD
تتعلم بناء خطوط أتمتة كاملة: من رفع الكود إلى نشره في الإنتاج بدون أي تدخل يدوي.
13
موصى بها بشدة
شهادة RHCE + Ansible على AWS
Red Hat Certified Engineer (RHCE) + Ansible
تجمع بين إدارة Linux على مستوى احترافي وأتمتة الإعدادات بـ Ansible. ظهرت في إعلانات وظائف مهندس الكلاود.
14
اختيارية
اختيارية: Azure Fundamentals (AZ-900)
OPTIONAL: Azure Fundamentals (AZ-900)
بعد إتمام كل ما سبق، إضافة اختيارية تُظهر معرفتك بمزوّد كلاود آخر. أضفها فقط بعد إتقان AWS.
🚀
بعد إنجاز هذا المسار
يمكنك البدء بتقديم طلبات التوظيف خلال ٦–٩ أشهر. بعد الخطوة ١١ (AWS SAA) أنت جاهز لبدء التقديم بثقة.

Git

Python / boto3

Docker

Kubernetes
إدارة وتشغيل مئات الحاويات تلقائياً.
Terraform
بناء البنية التحتية (سيرفرات، شبكات) عن طريق كتابة كود.
CI/CD

Ansible

YAML
لغة كتابة الإعدادات المستخدمة في Docker وKubernetes وCI/CD.
